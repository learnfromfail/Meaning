邊際利潤


沙漠裡的第一口水，是珍貴過第五十口水的，個肚飽和了。自助餐，每一款食啲，是好過只選擇一款的，因為會食得厭。 經濟學的＜邊際利潤＞（補位，物以罕為貴），新加坡總理某次面對在野黨派挑戰，于是說過：正正是大家覺得沒有問題，我才覺得問題很嚴重。現實生活上，以整體利益（公義）為大局，于是主動補位的人，不多。很多人只活在自己的岡位上。

編程和演算法


蘋果教父，喬布斯講過編程公民廿一世紀必須要學的知識之一，含有大量代碼上的管理和分配，定義工具。編程上，常有Don't repeat yourself的概念，編程這么好玩，是極具低重覆性的，新奇工作，雖然只是在技術上，但卻鍛練客觀思維，是不可或缺的。以下我用部分正確的Java來簡介一下，戰棋遊戲如ＦＦＴＡ２，的codes是基本的皮毛。

Class Role{ String RoleName = ""; Int HP = 100; Int MP = 100; Int X = 0; Int Y = 0; Attack(Role enemy){ enemy.HP = enemy.HP - 20; } Move(Int toX, Int toY){ this.X = toX; this.Y = toY; } Role(String Name, Int MaxHp, Int MaxMp){ this.RoleName = Name; this.Hp = MaxHp; this.Mp = MaxMp; } }//end Class
Class Director{ List[Role] AllRoles; List[Grid] AllGrids; Start(){ AllRoles.Add(new Role(“Eric”, 50, 20)); AllRoles.Add(new Role(“Mary”, 30, 80)); AllRoles.Add(new Role(“Mon1”, 500, 200)); AllRoles.Add(new Role(“Boss”, 500, 200)); For(true){ //Game Loop AllRoles[0].move(); AllRoles[1].move(); }//forLoop-end }//Start()-end }//Class Director-end
